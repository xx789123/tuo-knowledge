## 核心技能

### 技术方案结构（完整大纲）
```
一、项目概述
  1.1 项目背景
  1.2 建设目标
  1.3 建设范围

二、需求分析
  2.1 业务需求
  2.2 功能需求
  2.3 性能需求
  2.4 安全需求

三、技术架构设计
  3.1 总体架构
  3.2 技术选型
  3.3 网络架构
  3.4 安全架构

四、系统功能设计
  4.1 功能模块划分
  4.2 核心功能详细设计

五、实施方案
  5.1 项目组织
  5.2 实施计划
  5.3 质量保障
  5.4 风险管理

六、售后服务
  6.1 服务内容
  6.2 响应机制
  6.3 培训计划

七、案例与业绩
  7.1 类似项目案例
  7.2 团队资质证书
```

### 投标评分要点
| 评分维度 | 占比 | 应对策略 |
|---------|------|---------|
| 技术方案 | 40-50% | 完整、具体、可落地 |
| 报价 | 30-40% | 合理、有竞争力 |
| 业绩资质 | 10-20% | 提供真实案例 |
| 售后保障 | 5-10% | 响应快、承诺具体 |

### 政府采购新规（2025-2026）
- **双盲评审**：技术标与商务标分离评审
- **中小企业优惠**：价格扣除10-20%
- **国产化要求**：优先采购国产软件/设备
- **诚信记录**：失信企业限制投标
