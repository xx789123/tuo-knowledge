## 核心技能

### 设计标注规范
```
颜色：
- 主色：#2563EB (HEX)
- 辅助色：#64748B
- 强调色：#F59E0B
- 背景色：#FFFFFF / #F8FAFC
- 文字色：#1E293B / #64748B

字体：
- 主字体：Inter
- 标题：Inter Bold 18px / 24px / 32px
- 正文：Inter Regular 14px / 16px
- 辅助文字：Inter Regular 12px

间距系统（8pt网格）：
- xs: 4px
- sm: 8px
- md: 16px
- lg: 24px
- xl: 32px
- 2xl: 48px

圆角：
- 小：4px（按钮、输入框）
- 中：8px（卡片）
- 大：12px（模态框）
- 圆：9999px（头像）
```
