## 核心技能

### 项目结构
```
src/
├── api/          # API封装
├── assets/       # 静态资源
├── components/   # 公共组件
├── composables/  # 组合式函数
├── router/       # 路由配置
├── stores/       # 状态管理
├── types/        # TypeScript类型
└── views/        # 页面组件
```

### 组件规范
- TypeScript类型完整
- 单一职责
- 300行以内
- 统一错误处理
